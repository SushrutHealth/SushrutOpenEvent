AndStatus は、ツリー形式のスレッドで会話を表示する、低トラフィックなソーシャルネットワー キングのクライアントです。 GNU social ("StatusNet". Quitter, LoadAverage, Vinilox, など), Twitter および Pump.io を含む複数のプロトコルをサポートします。 AndStatus はあなたのアカウントをすべてのネットワークから 1 つのタイムラインにまとめて、オフライン中も読んだり投稿したりすることができます。

Android 4.1 以上。 オープンソースライセンス

AndStatus の特徴的な機能:

1. さまざまなソーシャルネットワー キング システムの多くのアカウント。 それぞれにいくつかのアカウントを持ち、任意の「あなた」として書き込み/返信し、アカウントとシステム間で共有することができます。
2. あなたの更新を投稿するためにオンラインにする必要はありません: 未送信の投稿はリブート後も ("再試行" キュー内に) 保持されます お使いのデバイスがオンラインのときに送信されます。
3. 便利なツリー形式の "会話ビュー"。
4. "グローバル検索" は、1 つのクエリですべての登録されているシステムの中の公開メッセージを検索できます。
5. お使いのデバイスの接続が良い場合、ツイート/メッセージ/アバター/添付画像はバック グラウンドで同期されます。
6. キャッシュされたデータは、数年または数日間のみ保存することができます - それはあなたの選択しだいです。
7. お使いのデバイス間で、アカウントやデータを簡単にバックアップおよび復元できます。
8. フリーで広告なし、コミュニティによるサポート。

AndStatus のすべての機能の詳細については、変更ログを参照してください。
